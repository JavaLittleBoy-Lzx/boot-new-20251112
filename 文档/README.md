## 🎯 快速开始

### 开发环境启动

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd boot-new-20251112
   ```

2. **配置数据库**
   - 创建数据库：`nefu_system`
   - 执行SQL脚本：`sql/add_visit_status_fields.sql`
   - 修改 `application.yml` 数据库配置

3. **启动应用**
   ```bash
   # 方式1：IDEA直接运行 ParkingManageApplication.java
   # 方式2：命令行
   mvn spring-boot:run
   ```

4. **访问服务**
   - 应用地址：http://www.xuerparking.cn:8675
   - API文档：http://www.xuerparking.cn:8675/doc.html
   - 健康检查：http://www.xuerparking.cn:8675/actuator/health

## 💡 技术支持

### 常见问题

1. **数据库连接失败**
   - 检查数据库是否启动
   - 验证用户名密码
   - 确认防火墙规则

2. **访客自动同步不工作**
   - 检查 `visitor.auto.enabled` 配置
   - 查看日志错误信息
   - 验证外部API可达性

3. **海康威视查询失败**
   - 验证 app-key 和 app-secret
   - 检查网络连接
   - 查看详细日志

**详细解决方案**：参见 [04-配置与部署指南 > 常见问题](./04-配置与部署指南.md#九常见问题)
