## 系统概述

### ✨ 功能特性

- ✅ **JWT Token认证**：基于JWT的无状态认证机制
- ✅ **BCrypt密码加密**：使用BCrypt算法加密存储密码
- ✅ **多角色权限管理**：支持admin、user、guest等多种角色
- ✅ **登录日志记录**：记录登录时间、IP、登录次数等
- ✅ **Token刷新机制**：支持Token过期刷新
- ✅ **用户信息管理**：完整的用户CRUD操作
- ✅ **软删除机制**：用户数据逻辑删除，可恢复
- ✅ **Swagger文档**：自动生成API文档

### 🏗️ 技术栈

- **Spring Boot**: 2.5.5
- **MyBatis Plus**: 3.4.3
- **JWT**: auth0 java-jwt 3.18.2
- **Spring Security Crypto**: 密码加密
- **MySQL**: 5.7+
- **Lombok**: 简化代码
- **Swagger**: API文档生成

## 数据库配置

### 1. 执行SQL脚本

在MySQL中执行 `sql/sys_user.sql` 文件：

```bash
mysql -u root -p your_database < sql/sys_user.sql
```

或者在MySQL客户端中直接运行：

```sql
source /path/to/sql/sys_user.sql
```

### 2. 数据库表结构

**表名**: `sys_user`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，自增 |
| username | VARCHAR(50) | 用户名，唯一 |
| password | VARCHAR(255) | 密码（BCrypt加密） |
| real_name | VARCHAR(50) | 真实姓名 |
| nickname | VARCHAR(50) | 昵称 |
| email | VARCHAR(100) | 邮箱，唯一 |
| phone | VARCHAR(20) | 手机号，唯一 |
| role | VARCHAR(20) | 角色（admin/user/guest） |
| status | TINYINT | 状态（0-禁用，1-启用） |
| avatar | VARCHAR(255) | 头像URL |
| gender | TINYINT | 性别（0-未知，1-男，2-女） |
| department | VARCHAR(100) | 部门 |
| position | VARCHAR(100) | 职位 |
| permissions | TEXT | 权限（JSON数组） |
| last_login_time | DATETIME | 最后登录时间 |
| last_login_ip | VARCHAR(50) | 最后登录IP |
| login_count | INT | 登录次数 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
| create_by | VARCHAR(50) | 创建人 |
| update_by | VARCHAR(50) | 更新人 |
| is_deleted | TINYINT | 是否删除（0-否，1-是） |
| remark | VARCHAR(500) | 备注 |

| 用户名 | 密码 | 角色 | 权限 | 说明 |
|--------|------|------|------|------|
| admin | 123456 | admin | ["*"] | 管理员，拥有所有权限 |
| user | 123456 | user | ["read", "write"] | 普通用户 |

### 2. Application配置

在 `application.yml` 或 `application.properties` 中配置：

```yaml
# JWT配置
jwt:
  secret: parkingSystemSecretKeyForJwtTokenGeneration2024  # JWT密钥，生产环境请修改
  expiration: 86400  # Token过期时间（秒），默认24小时

# 数据库配置
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/your_database?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
    driver-class-name: com.mysql.cj.jdbc.Driver

# MyBatis Plus配置
mybatis-plus:
  mapper-locations: classpath:mapper/*.xml
  type-aliases-package: com.parkingmanage.entity
  configuration:
    map-underscore-to-camel-case: true
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
```

### 基础URL

```
http://10.100.111.2:8675/api/auth
```

### 1. 用户登录

**接口**: `POST /api/auth/login`

**请求头**: 
```
Content-Type: application/json
```

**请求体**:
```json
{
  "username": "admin",
  "password": "123456",
  "rememberMe": false
}
```

**响应成功** (200):
```json
{
  "code": "0",
  "msg": "成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "tokenType": "Bearer",
    "expiresIn": 86400,
    "user": {
      "id": 1,
      "username": "admin",
      "name": "系统管理员",
      "nickname": "管理员",
      "role": "admin",
      "email": "admin@nefu.edu.cn",
      "phone": "13800138000",
      "avatar": null,
      "department": "技术部",
      "position": "系统管理员",
      "permissions": ["*"],
      "loginTime": "2024-12-06T14:30:00"
    }
  }
}
```

**响应失败** (401):
```json
{
  "code": "401",
  "msg": "用户名或密码错误",
  "data": null
}
```

**接口**: `POST /api/auth/logout`

**请求头**:
```

```

**响应成功** (200):
```json

  "msg": "成功",
  "data": "登出成功"
}
```

### 3. 刷新Token

**接口**: `POST /api/auth/refresh`

**请求头**:
```
Authorization: Bearer {token}
```

**响应成功** (200):
```json
{
  "code": "0",

  "data": {
```

**接口**: `GET /api/auth/validate`

**请求头**:

**响应成功** (200):

```json
  "code": "0",
  "msg": "成功",
  "data": {

    "message": "Token有效"
```

**接口**: `GET /api/auth/userinfo`

**请求头**:

```json
{
  "code": "0",
```

## 常见问题

### Q1: Token一直提示无效？

**A**: 检查以下几点：
1. JWT密钥配置是否正确（`jwt.secret`）
2. Token是否正确传递（格式：`Bearer {token}`）
3. Token是否已过期
4. 数据库中用户状态是否为启用（status=1）

### Q2: 密码验证失败？

**A**: 确认：
1. 数据库中的密码是否使用BCrypt加密
2. 原始密码是否正确（初始密码都是123456）
3. 密码加密时的salt是否一致

### Q3: 如何修改JWT过期时间？

**A**: 在`application.yml`中修改：
```yaml
jwt:
  expiration: 604800  # 7天（秒）
```

### Q4: 如何生成新的BCrypt密码？

**A**: 使用以下Java代码：
```java
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;

BCryptPasswordEncoder encoder = new BCryptPasswordEncoder();
String encodedPassword = encoder.encode("your_password");
System.out.println(encodedPassword);
```

### Q5: 如何添加自定义权限？

**A**: 修改用户的permissions字段：
```sql
UPDATE sys_user 
SET permissions = '["read", "write", "delete", "custom_permission"]' 
WHERE username = 'admin';
```

### Q6: 数据库连接失败？

**A**: 检查：
1. MySQL服务是否启动
2. 数据库名称是否正确
3. 用户名密码是否正确
4. 防火墙是否允许连接

### Q7: Mapper找不到？

**A**: 确认：
1. `@MapperScan`注解是否正确配置
2. Mapper接口是否添加`@Mapper`注解
3. MyBatis Plus配置是否正确

### Q8: 如何查看SQL日志？

**A**: 在`application.yml`中添加：
```yaml
mybatis-plus:
  configuration:
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
```

## 🔒 安全建议

1. **生产环境配置**：
   - 修改JWT密钥为复杂的随机字符串
   - 使用环境变量存储敏感配置
   - 启用HTTPS加密传输

2. **密码策略**：
   - 强制用户修改初始密码
   - 实施密码复杂度要求
   - 定期提醒用户更换密码

3. **Token安全**：
   - 合理设置Token过期时间
   - 实现Token黑名单机制
   - 限制Token刷新次数

4. **访问控制**：
   - 实施IP白名单
   - 添加登录失败次数限制
   - 记录所有敏感操作日志
